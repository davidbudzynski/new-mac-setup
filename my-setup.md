# Setting up macOS for web development and data science

## Homebrew

Instal Homebrew and and update it.

``` sh
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

```sh
brew update
```
